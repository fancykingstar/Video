## Requirements

You need the following to run this app:

* Node v8.x or higher
* NPM v5.x or higher
* MAC OR Ubuntu

## Setup

Open a terminal at the repo root, and run the following:

1. install now cli 
     npm i -g now
2. install packages
    yarn install
3. run project
    sudo now dev

Your app will be available at http://localhost:3000.
